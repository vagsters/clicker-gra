# Offline Clicker

Prosta gra typu clicker, która działa całkowicie offline i nie wymaga backendu. Postęp gry zapisywany jest w `localStorage` przeglądarki.

## Uruchomienie

1. Skopiuj repozytorium lub pobierz paczkę ZIP.
2. Otwórz plik `index.html` w przeglądarce.

## Rozszerzanie gry

Logika gry jest zaimplementowana w pliku `script.js`. Wszystkie ulepszenia definiowane są w tablicy `upgradeDefs`, co ułatwia dodawanie nowych elementów rozgrywki.
